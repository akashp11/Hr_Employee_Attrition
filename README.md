# Hr_Employee_Attrition
Data Anylist Project For Hr_Employee_Attrition
Business Problem

Employee attrition leads to increased recruitment costs, productivity loss, and organizational instability. Understanding why employees leave is critical for improving retention strategies. This dashboard helps HR teams identify trends, risk factors, and high-attrition segments.

Objectives

Analyze overall employee attrition rate

Identify key drivers of attrition

Compare attrition across departments, roles, and demographics

Support data-driven HR retention strategies

Tools & Technologies Used

Power BI Desktop – Dashboard development and visualization

Power Query – Data cleaning and transformation

DAX (Data Analysis Expressions) – KPIs and calculated measures

Excel / CSV – HR data source

Key KPIs

Total Employees

Attrition Count

Attrition Rate (%)

Average Age

Average Monthly Income

Average Years at Company

Dashboard Features

Overall Attrition Overview – High-level attrition metrics

Department-wise Attrition – Attrition by department and job role

Age Group Analysis – Attrition by employee age groups

Gender Analysis – Attrition comparison by gender

Salary & Experience Impact – Attrition vs income and tenure

Interactive Filters & Slicers – Dynamic analysis by category

Data Preparation Steps

Imported HR employee data into Power BI

Cleaned and formatted data using Power Query

Created calculated columns and measures using DAX

Built relationships (if applicable)

Designed interactive visuals and KPIs

Visuals Used

KPI cards

Bar and column charts

Donut and pie charts

Line charts

Tables and matrices

Slicers for filtering

How to Use This Project

Clone or download this GitHub repository

Open Hr_Employee_Attrition.pbix in Power BI Desktop

Refresh the dataset if required

Use slicers to explore attrition insights

Analyze patterns to derive HR recommendations
